# self_radio_posting

글을 작성할 수 있는 게시판입니다.  
댓글과 답글을 작성할 수 있으며 통합 검색을 통해 작성된 게시글을 검색할 수 있습니다.

  
이미지와 글을 함께 올리면 이미지에 자막이 추가된 비디오가 생성됩니다. 구글 tts api를 사용하여 자막이 비디오의 음성으로 재생됩니다. 

아래 주소로 접속할 수 있습니다.
http://43.202.234.145:8080/

사용한 기술 스택
- JAVA spring boot
- JavaScript
- Amazon s3 저장소
- Postgres 데이터 베이스
- Google text-to-speech api
- Amazon Lightsail 서버
